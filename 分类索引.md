# 文档类型分类索引

```
docx-dfgp-knowledge/
├── templates/

│   ├── letter/         # 函

│   ├── report/         # 报告
│   ├── announcement/   # 公告
│   ├── notice/         # 通告
│   ├── proposal/       # 议案
│   ├── resolution/     # 决议
│   ├── directive/      # 指示

│   ├── academic/       # 学术论文
│   └── student/        # 学生作业
```

## 公文类型定义 (GB/T 9704-2012)

| 类型 | 识别依据 | 识别置信度阈值 |
|------|----------|---------------|
| **通知** | 标题含"通知"、有主送机关 | ≥90% 自动 |
| **决定** | 正文含"经研究决定" | ≥90% 自动 |
| **函** | 平行文、无上下级隶属关系 | ≥90% 自动 |
| **批复** | 回应请示、含"同意"/"不同意" | ≥90% 自动 |
| **命令(令)** | 标题含"命令"/"令"字 | ≥90% 自动 |
| **通报** | 含"通报批评"/"表彰" | ≥90% 自动 |
| **报告** | 上行文、含"请审阅"/"请批示" | ≥90% 自动 |
| **公告** | 发布范围广泛、无主送机关 | ≥90% 自动 |
| **通告** | 事项性质面向社会 | ≥90% 自动 |
| **议案** | 提交人大审议 | ≥90% 自动 |
| **决议** | 含"会议决定" | ≥90% 自动 |
| **指示** | 含安排部署性质 | ≥90% 自动 |
| **条例** | 法规性质、标题含"条例" | ≥90% 自动 |
| **纪要** | 标题含"纪要"、有出席/请假人员 | ≥90% 自动 |

## 审计维度定义

### 精确匹配 (FATAL)

| 字段 | 验证方法 | 容差 |
|------|----------|------|
| 发文字号 | exact_match | 0 |
| 成文日期 | exact_match | 0 |
| 发文机关标志 | exact_match | 0 |

### 实体保留 (CRITICAL)

| 字段 | 验证方法 | 容差 |
|------|----------|------|
| 机关名称 | entity_preserve | 0 |
| 附件名称 | entity_preserve | 0 |
| 主送机关列表 | list_complete | 0 |

### 数值精度 (MAJOR)

| 字段 | 验证方法 | 容差 |
|------|----------|------|
| 总字数 | word_count | 5% |
| 金额/百分比 | numeric_preserve | 0 |
| 表格数量 | count_match | 0 |
| 图片数量 | count_match | 0 |

### 格式规范 (参考SRS)

| 检查项 | 要求 |
|--------|------|
| 纸张尺寸 | A4 (210mm × 297mm) |
| 页边距 | 上37mm/下35mm/左28mm/右28mm |
| 字体 | 仿宋_GB2312, 3号(16pt) |
| 行数 | 22行/页 |
| 字数 | 28字/行 |

## 置信度阈值策略

| 置信度 | 处理方式 |
|--------|----------|
| ≥ 90% | 全自动处理 |
| 80-90% | 自动处理 + 标记待确认 |
| < 80% | 暂停，等待人工确认 |
