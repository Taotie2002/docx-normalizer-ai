## 识别特征

- **标题**：论文标题（无"关于...的通知"格式），居中黑体加粗
- **摘要**：以"摘　要"开头，正文楷体11pt
- **关键词**：以"关键词："开头，黑体11pt加粗
- **节编号**：使用阿拉伯数字节编号，如"1"、"2.1"、"3.2.1"
- **参考文献**：以"[序号] 作者"格式，以Times New Roman 10pt呈现
- **无公文特征**：无发文字号、无落款日期格式（日期在作者单位中）、无"妥否请批示"

## 段落识别函数

```python
import re

def is_paper_title(text, index):
    """论文标题：居中，中文，黑体加粗，无'关于'"""
    if index == 0 and '关于' not in text and len(text) <= 40 and len(text) > 5:
        return True
    return False

def is_abstract(text):
    """摘要标题"""
    return text.strip() == "摘　要" or text.strip() == "摘要"

def is_keywords(text):
    """关键词"""
    return text.startswith("关键词") or text.startswith("关键词：")

def is_section_heading(text):
    """节标题：如'1　引言'、'2.1　方法'"""
    # 匹配 数字 + 空格 + 汉字
    return bool(re.match(r'^\d+　[\u4e00-\u9fff]', text))

def is_reference_section(text):
    """参考文献"""
    return "参考文献" in text

def is_reference_item(text):
    """参考文献条目：[1] 开头"""
    return bool(re.match(r'^\[\d+\]', text.strip()))
```
